# Siyuan Yang — Academic Homepage

A lightweight, responsive academic homepage for GitHub Pages.

## Deploy

1. Create a GitHub repository named `YOUR_USERNAME.github.io`.
2. Upload all files in this folder to the repository root.
3. In **Settings → Pages**, choose **Deploy from a branch**, then select the default branch and `/ (root)`.
4. The site will appear at `https://YOUR_USERNAME.github.io/`.

## Personalize

- Replace `assets/profile-placeholder.svg` with your photo and update the image path in `index.html`, or overwrite it while keeping the same filename.
- Export your latest CV as `assets/Siyuan_Yang_CV.pdf` so that the **CV** button works.
- Add project, paper, GitHub, DOI, or arXiv links inside each publication entry as needed.
- Update the short bio and research interests directly in `index.html`.

The current publication list excludes submitted, under-review, and under-revision manuscripts.
